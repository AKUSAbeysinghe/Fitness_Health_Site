# Health & Wellness Directory

Simple React + Tailwind project for browsing health & wellness categories.

## Categories

- Pharmacies / Drugstores: Walgreens, CVS, Boots  
- Fitness & Wellness Retail: GNC, Vitamin Shoppe  
- Gyms / Fitness Clubs: Planet Fitness, Anytime Fitness  

## Setup

npm install  
npm run dev  

## Tech
React + Tailwind CSS + PHP
